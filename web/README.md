# Palliative Care Hub

ระบบบริหารจัดการผู้ป่วย palliative care สำหรับงานเยี่ยมบ้านและสรุปผู้บริหาร

## จุดเด่น

- ดึงผู้ป่วยเข้าทะเบียนจาก HOSXP ตามรหัสโรคและเงื่อนไขพื้นที่
- เก็บทะเบียนแยกในฐานข้อมูล `paliative` เพื่อไม่กระทบฐาน `hos`
- มีปฏิทินเยี่ยมบ้าน, ปุ่มเช็คเยี่ยม, เปลี่ยนวันนัด, และจำหน่ายครบช่วง/เสียชีวิต
- รองรับ responsive web ใช้งานบน iOS / Android ผ่าน browser ได้ทันที

## วิธีรัน

1. คัดลอก `.env.example` เป็น `.env.local`
2. ปรับค่าฐานข้อมูลจริง
3. สร้าง schema จาก `database/paliative.sql`
4. รัน `npm run dev`

## หมายเหตุ

ถ้ายังไม่ได้ตั้งค่าฐานข้อมูล แอปจะใช้ข้อมูลตัวอย่างในตัวเพื่อให้ demo UI และ flow ได้ก่อน
